# Forecast Wizard V4 - Complete Specification

## Overview

A full-screen, guided forecast builder that walks clients through the 7-step CFO methodology for building financial forecasts. The wizard uses Xero data (or uploaded CSV) as a foundation and provides AI assistance throughout.

**Key Principles:**
- P&L is the main canvas (full-width display)
- Guided steps with AI assistant available for help
- Supports 3-year forecasting: Monthly for Y1, Quarterly for Y2-3
- Desktop-only (no mobile requirement)
- Professional CFO experience in language clients understand

### Components:

1. **Step Bar (Top)**: Progress indicator showing all 7 steps, current step highlighted, completed steps with checkmarks

2. **Year Tabs**: Switch between FY26 (monthly), FY27 (quarterly), FY28 (quarterly). Only relevant tabs shown per step.

3. **P&L Canvas (Main Area)**: Full-width display of current step's data. Changes based on active step and selected year tab.

4. **3-Year Summary (Bottom)**: Always-visible summary showing key metrics across all 3 years:
   - Revenue trajectory
   - Gross Profit % trajectory
   - Net Profit trajectory
   - Updates in real-time as user makes changes

5. **AI Assistant (Floating Panel)**:
   - Collapsible floating panel (bottom-right)
   - Context-aware based on current step
   - Can be minimized to icon
   - Provides suggestions, answers questions, helps with decisions

## Technical Notes

### API Endpoints Required

```
GET  /api/goals?business_id=X                    - Fetch existing goals
GET  /api/Xero/pl-summary?business_id=X          - Prior year P&L
GET  /api/Xero/employees?business_id=X           - Team from Xero
GET  /api/strategic-initiatives?business_id=X   - For CapEx linkage
POST /api/forecast-wizard/save                   - Save draft
POST /api/forecast-wizard/generate               - Generate final forecast
GET  /api/forecast-wizard/:id                    - Load existing draft
```

### Calculations

**Superannuation**: 11.5% of salary (FY2025 rate, increases annually)
**Contractor**: No super, cost = annual amount / 12

**Commission Calculation**:
```
monthlyCommission = (monthlyRevenue[lineId] * commissionPct) / 100
```

**Seasonal Revenue Distribution**:
```
monthlyRevenue[month] = annualTarget * (priorYearMonthly[month] / priorYearTotal)
```

**Straight-Line Distribution**:
```
monthlyRevenue = annualTarget / 12
quarterlyRevenue = annualTarget / 4
```
