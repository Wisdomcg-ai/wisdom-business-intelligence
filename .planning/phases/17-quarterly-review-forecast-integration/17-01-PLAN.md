Create `src/app/api/forecast/quarterly-summary/route.ts` following the actuals-summary pattern:

**Query params:** `forecastId` (required), `quarter` (1-4, required), `fiscalYear` (required), `yearStartMonth` (optional, default 7)

**Implementation:**
1. `export const dynamic = 'force-dynamic'`
2. Import `createRouteHandlerClient` from `@/lib/supabase/server`
3. Import `getMonthKeysForQuarter`, `sumMonthsForKeys` from `@/lib/utils/fiscal-year-utils`
4. Auth check: `supabase.auth.getUser()` — return 401 if not authenticated
5. Parse and validate query params from `request.url` via `new URL(request.url).searchParams`:
   - `forecastId`: string, required (400 if missing)
   - `quarter`: parse as int, must be 1-4 (400 if invalid)
   - `fiscalYear`: parse as int, required (400 if missing)
   - `yearStartMonth`: parse as int, default 7
6. Fetch forecast row: `supabase.from('financial_forecasts').select('id, business_id, fiscal_year, is_locked').eq('id', forecastId).maybeSingle()` — return 404 if not found
7. Fetch all forecast_pl_lines: `supabase.from('forecast_pl_lines').select('account_name, account_type, category, forecast_months, actual_months').eq('forecast_id', forecastId)`
8. Get quarter month keys: `const qKeys = getMonthKeysForQuarter(quarter, fiscalYear, yearStartMonth)`
9. Copy `isRevenue()` and `isCOGS()` classification functions from actuals-summary/route.ts (same REVENUE_CATEGORIES and COGS_CATEGORIES arrays)
10. Aggregate by category:
    - Revenue lines: where `isRevenue(line.category, line.account_type)`
    - COGS lines: where `isCOGS(line.category)`
    - OpEx lines: everything else (not revenue, not COGS)
    - For each category group, sum `forecast_months` and `actual_months` using `sumMonthsForKeys`
11. Calculate derived values:
    - `grossProfit = { forecast: revForecast - cogsForecast, actual: revActual - cogsActual }`
    - `netProfit = { forecast: gpForecast - opexForecast, actual: gpActual - opexActual }`
    - For each metric: `variance = actual - forecast`, `variancePct = forecast !== 0 ? Math.round((variance / forecast) * 100) : 0`
12. `hasActuals`: true if ANY actual_months key for the quarter has a non-zero value across all lines
13. Return JSON response shape:
```typescript
{
  quarter, fiscalYear, forecastId,
  revenue: { forecast, actual, variance, variancePct },
  cogs: { forecast, actual, variance, variancePct },
  grossProfit: { forecast, actual, variance, variancePct },
  opex: { forecast, actual, variance, variancePct },
  netProfit: { forecast, actual, variance, variancePct },
  hasActuals: boolean
}
```

Use `console.error('[quarterly-summary]', ...)` for error logging (matching actuals-summary pattern). Return `NextResponse.json({ error: ... }, { status: ... })` on all error paths.
