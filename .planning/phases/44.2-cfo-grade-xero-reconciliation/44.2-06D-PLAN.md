<objective>
Path A — Balance Sheet sync. Mirrors 06B's per-month single-period discipline but for BS.

Per Calxa-via-Cowork Q1 (Xero developer docs sourced): Reports/BalanceSheet has the same documented
periods-parameter bug as Reports/ProfitAndLoss. We avoid it the same way: one query per as-of date,
no periods parameter. Each query returns the canonical Xero web BS for that point-in-time.

Per Calxa-via-Cowork Q2: Reports/BalanceSheet already includes FX revaluation, system-maintained accounts
(Retained Earnings, Current Year Earnings, Bank Revaluation, Currency Revaluation Reserve), foreign-currency
AR/AP — all the things that would otherwise need a TrialBalance reconciliation pass. So no separate
TrialBalance fetch is needed in production. (06E adds TrialBalance ONLY as a verification gate.)

Per Calxa-via-Cowork Q3 verification gate #4 (Net Assets == Equity): BS in balance is a fundamental
accounting invariant. We assert this per balance_date as a runtime gate. If a Xero BS query violates
it (rare but possible during ongoing transactions or edge cases), mark the tenant 'partial' and alert.

Output: 1 new parser module + extended orchestrator + 2 test files. Reuses 06B's xero-api-client,
accounts-catalog, organisation modules. No new DB schema (06C handles that).
</objective>

    ```ts
    function classifyBS(sectionTitle: string): 'asset' | 'liability' | 'equity' | null {
      const t = sectionTitle.toLowerCase()
      if (t.includes('asset')) return 'asset'
      if (t.includes('liabilit')) return 'liability'
      if (t.includes('equity')) return 'equity'
      return null
    }
    ```

    Carry-forward logic mirrors PL (D-44.2-14):
    ```ts
    let currentParentType: 'asset' | 'liability' | 'equity' | null = null
    let currentSectionTitle: string | null = null
    function walk(rows, depth) {
      for (const row of rows) {
        if (row.RowType === 'Section') {
          const title = row.Title?.trim() ?? ''
          const titleType = classifyBS(title)
          if (title && titleType) {
            currentParentType = titleType
            currentSectionTitle = null  // top-level section reached, sub-section context resets
          } else if (title) {
            currentSectionTitle = title  // sub-section captured as section field
          }
          if (row.Rows) walk(row.Rows, depth + 1)
        } else if (row.RowType === 'Row' && currentParentType) {
          // emit ParsedBSRow with currentParentType + currentSectionTitle
        }
      }
    }
    ```
