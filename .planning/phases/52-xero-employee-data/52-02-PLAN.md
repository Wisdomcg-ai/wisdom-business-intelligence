<objective>
Ship the re-import / reconciliation flow on Step 4 — XERO-S4-05's "re-import without losing manual edits" requirement, implementing **Operator's Option D** end-to-end. After this plan lands, the operator can:

1. After an initial import (52-01), see a new **"Refresh from Xero"** button on Step 4 (visible only when at least one row has `_xeroFingerprint`)
2. Click it → wizard re-fetches `/api/Xero/employees` → reconciliation modal opens showing:
   - Per-row diff for any matched employee where Xero values have changed AND the operator has overridden that field (CONFLICT — operator decides)
   - Silent updates for fields the operator has NOT touched (no prompt — just applied)
   - "New from Xero" section listing employees in Xero that don't yet have a wizard row (operator opts in)
3. Per-field decisions: **[Keep yours] [Accept Xero] [Edit]** — fingerprint updated either way so the same conflict doesn't re-prompt next refresh
4. Bulk actions: **Accept all Xero changes** + **Keep all my changes**
5. Click **Apply N changes** to commit; modal closes; wizard state reflects the reconciled values
6. If nothing has changed, modal opens briefly with "Everything is in sync with Xero" message — no awkward silent no-op

**The core algorithm** (per 52-RESEARCH.md "Re-import provenance tracking" + Operator's Option D):

```
For each xeroEmployee in fresh fetch:
  match = findMatchingTeamMember(xeroEmployee, state.teamMembers)
  if no match:
    add to "New from Xero" candidates
    continue
  for each Xero-tracked field {payFrequency, standardHours, hourlyRate, currentSalary, name, role, type}:
    currentValue = match[field]
    lastImportedValue = match._xeroFingerprint?[field]
    newXeroValue = enriched[field]
    operatorOverrode = match._overriddenFields?.includes(field)
    xeroChanged = newXeroValue !== lastImportedValue
    if !xeroChanged:
      skip (UNCHANGED)
    elif xeroChanged AND !operatorOverrode:
      silently apply newXeroValue + update fingerprint (UPDATED-BY-XERO-ONLY)
    else:  # xeroChanged AND operatorOverrode
      add to conflicts list (CONFLICT — operator decides)
```

**Critical safety guarantee** (52-RESEARCH.md Pitfall 6): the algorithm iterates `xeroEmployees`, NEVER `state.teamMembers`. Manually-added members (no `_xeroEmployeeId`) are completely untouchable by re-import. No `removeTeamMember` calls in any code path of this plan.

**No new endpoints. No DB changes. No rollup math edits.** This plan is pure UI + pure provenance helpers on top of the API + types from 52-00 and the override-tracking from 52-01.

## New helpers in xero-payroll-mapping.ts (additive — extend the module from 52-00 + 52-01)

```typescript
import type { TeamMember, NewHire, XeroFieldFingerprint } from '../types';

/**
 * Match a Xero employee to an existing wizard TeamMember.
 * Strategy (in order of confidence):
 *   1. Exact match on _xeroEmployeeId (highest — survives name changes)
 *   2. Case-insensitive email match (if both present)
 *   3. Case-insensitive full-name match
 *   4. No match → returns undefined (caller adds as new)
 *
 * NOTE: Caller must iterate xeroEmployees and look up against teamMembers,
 * NOT the reverse (per 52-RESEARCH Pitfall 6 — manually-added members must
 * not be filtered out by this function).
 */
export function findMatchingTeamMember(
  xeroEmp: { employee_id: string; email?: string; full_name: string },
  teamMembers: Array<Pick<TeamMember, 'id' | '_xeroEmployeeId' | 'name'> & { email?: string }>,
): TeamMember['id'] | undefined {
  // Tier 1: _xeroEmployeeId
  const byId = teamMembers.find(m => m._xeroEmployeeId === xeroEmp.employee_id);
  if (byId) return byId.id;
  // Tier 2: email
  if (xeroEmp.email) {
    const xeroEmail = xeroEmp.email.toLowerCase();
    const byEmail = teamMembers.find(m => (m as any).email?.toLowerCase() === xeroEmail);
    if (byEmail) return byEmail.id;
  }
  // Tier 3: full name
  const xeroName = xeroEmp.full_name.toLowerCase().trim();
  const byName = teamMembers.find(m => m.name.toLowerCase().trim() === xeroName);
  if (byName) return byName.id;
  return undefined;
}

/**
 * The 7 fields tracked by Xero re-import reconciliation.
 * Source of truth — keep in sync with markFieldOverridden / Step4Team usage.
 */
export const XERO_TRACKED_FIELDS = [
  'name', 'role', 'type', 'payFrequency', 'standardHours', 'hourlyRate', 'currentSalary',
] as const;
export type XeroTrackedField = typeof XERO_TRACKED_FIELDS[number];

export type FieldDiffVerdict = 'unchanged' | 'updated-by-xero-only' | 'conflict';

export interface FieldDiff {
  field: XeroTrackedField;
  currentValue: unknown;
  lastImportedValue: unknown;
  newXeroValue: unknown;
  verdict: FieldDiffVerdict;
}

export interface MemberDiff {
  memberId: TeamMember['id'];
  xeroEmployeeId: string;
  fields: FieldDiff[];
}

/**
 * Compute a per-field diff between the current wizard member, its last-imported
 * fingerprint, and the fresh Xero values. Returns the field list with verdicts.
 *
 * verdict logic:
 *   xeroChanged = newXeroValue !== lastImportedValue
 *   operatorOverrode = member._overriddenFields?.includes(field)
 *   - !xeroChanged → 'unchanged'
 *   - xeroChanged && !operatorOverrode → 'updated-by-xero-only' (silent apply)
 *   - xeroChanged && operatorOverrode → 'conflict' (operator decides)
 */
export function computeReconciliationDiff(
  member: TeamMember,
  freshXeroValues: Pick<TeamMember, XeroTrackedField>,
): MemberDiff {
  const fingerprint = member._xeroFingerprint ?? {};
  const overridden = new Set(member._overriddenFields ?? []);
  const fields: FieldDiff[] = XERO_TRACKED_FIELDS.map(field => {
    const currentValue = (member as any)[field];
    const lastImportedValue = (fingerprint as any)[field];
    const newXeroValue = (freshXeroValues as any)[field];
    const xeroChanged = !valuesEqual(newXeroValue, lastImportedValue);
    const operatorOverrode = overridden.has(field);
    let verdict: FieldDiffVerdict;
    if (!xeroChanged) verdict = 'unchanged';
    else if (!operatorOverrode) verdict = 'updated-by-xero-only';
    else verdict = 'conflict';
    return { field, currentValue, lastImportedValue, newXeroValue, verdict };
  });
  return { memberId: member.id, xeroEmployeeId: member._xeroEmployeeId!, fields };
}

function valuesEqual(a: unknown, b: unknown): boolean {
  // Treat undefined and null as equivalent (both = "absent")
  if (a == null && b == null) return true;
  if (a == null || b == null) return false;
  if (typeof a === 'number' && typeof b === 'number') {
    // 2-decimal tolerance for floats (hourly rates, hours)
    return Math.abs(a - b) < 0.005;
  }
  return a === b;
}

export type ReconciliationDecision = 'accept-xero' | 'keep-mine' | 'edit';

/**
 * Compute the partial member update to apply for a single field decision.
 *
 *   accept-xero  → set field to newXeroValue + remove field from _overriddenFields + update _xeroFingerprint[field]
 *   keep-mine    → leave field alone + update _xeroFingerprint[field] (so future re-imports don't re-prompt)
 *                  + ensure field is IN _overriddenFields (it already is for a 'conflict' verdict, but idempotent)
 *   edit (with explicit value) → set field to operatorValue + ADD field to _overriddenFields + update _xeroFingerprint[field]
 */
export function applyReconciliationDecision(
  member: TeamMember,
  field: XeroTrackedField,
  decision: ReconciliationDecision,
  newXeroValue: unknown,
  operatorValue?: unknown,
): Partial<TeamMember> {
  const updatedFingerprint: XeroFieldFingerprint = {
    ...(member._xeroFingerprint ?? {}),
    [field]: newXeroValue as any,
  };
  const currentOverrides = member._overriddenFields ?? [];

  if (decision === 'accept-xero') {
    return {
      [field]: newXeroValue,
      _xeroFingerprint: updatedFingerprint,
      _overriddenFields: currentOverrides.filter(f => f !== field),
      _xeroImportedAt: new Date().toISOString(),
    } as Partial<TeamMember>;
  }
  if (decision === 'keep-mine') {
    return {
      _xeroFingerprint: updatedFingerprint,
      _overriddenFields: currentOverrides.includes(field) ? currentOverrides : [...currentOverrides, field],
      _xeroImportedAt: new Date().toISOString(),
    } as Partial<TeamMember>;
  }
  // edit
  return {
    [field]: operatorValue,
    _xeroFingerprint: updatedFingerprint,
    _overriddenFields: currentOverrides.includes(field) ? currentOverrides : [...currentOverrides, field],
    _xeroImportedAt: new Date().toISOString(),
  } as Partial<TeamMember>;
}

/**
 * Apply ALL silent (updated-by-xero-only) field updates for a member in one shot.
 * Returns the partial member update to pass to actions.updateTeamMember.
 * Used immediately after computeReconciliationDiff — silent updates apply
 * before the modal renders so the operator only sees genuine conflicts.
 */
export function applySilentXeroUpdates(
  member: TeamMember,
  diff: MemberDiff,
): Partial<TeamMember> | null {
  const silent = diff.fields.filter(f => f.verdict === 'updated-by-xero-only');
  if (silent.length === 0) return null;
  const update: any = { _xeroImportedAt: new Date().toISOString() };
  const fingerprint: XeroFieldFingerprint = { ...(member._xeroFingerprint ?? {}) };
  for (const f of silent) {
    update[f.field] = f.newXeroValue;
    (fingerprint as any)[f.field] = f.newXeroValue;
  }
  update._xeroFingerprint = fingerprint;
  return update as Partial<TeamMember>;
}

// State for reconciliation flow
const [showReconcile, setShowReconcile] = useState(false);
const [reconcileLoading, setReconcileLoading] = useState(false);
const [reconcileError, setReconcileError] = useState<string | null>(null);
const [conflicts, setConflicts] = useState<MemberDiff[]>([]);  // only members with >=1 conflict field
const [silentUpdates, setSilentUpdates] = useState<Array<{ memberId: string; update: Partial<TeamMember> }>>([]);
const [newFromXero, setNewFromXero] = useState<XeroEmployeeApiShape[]>([]);
const [selectedNewIds, setSelectedNewIds] = useState<Set<string>>(new Set());
const [pendingDecisions, setPendingDecisions] = useState<Record<string, Record<string, ReconciliationDecision>>>({});
// pendingDecisions[memberId][field] = decision

// Visibility: only show Refresh button if any row has _xeroFingerprint
const hasAnyXeroSourcedRow = useMemo(
  () => state.teamMembers.some(isXeroSourcedRow) || (state.newHires ?? []).some(isXeroSourcedRow),
  [state.teamMembers, state.newHires],
);

const openReconcile = async () => {
  setReconcileLoading(true);
  setReconcileError(null);
  setShowReconcile(true);
  try {
    const res = await fetch(`/api/Xero/employees?business_id=${businessId}`);
    if (!res.ok) {
      setReconcileError('Failed to fetch Xero employees');
      return;
    }
    const data = await res.json();
    const xeroEmps: XeroEmployeeApiShape[] = data.employees ?? [];

    const newConflicts: MemberDiff[] = [];
    const newSilent: typeof silentUpdates = [];
    const newCandidates: XeroEmployeeApiShape[] = [];

    for (const xeroEmp of xeroEmps) {
      const matchedId = findMatchingTeamMember(xeroEmp, state.teamMembers);
      if (!matchedId) {
        newCandidates.push(xeroEmp);
        continue;
      }
      const member = state.teamMembers.find(m => m.id === matchedId)!;
      const enriched = enrichWizardMemberFromXeroEmployee(xeroEmp);
      const freshValues = {
        name: enriched.name,
        role: enriched.role,
        type: enriched.type,
        payFrequency: enriched.payFrequency,
        standardHours: enriched.standardHours,
        hourlyRate: enriched.hourlyRate,
        currentSalary: enriched.currentSalary,
      };
      const diff = computeReconciliationDiff(member, freshValues as any);
      const hasConflict = diff.fields.some(f => f.verdict === 'conflict');
      if (hasConflict) newConflicts.push(diff);
      const silentUpdate = applySilentXeroUpdates(member, diff);
      if (silentUpdate) newSilent.push({ memberId: member.id, update: silentUpdate });
    }

    setConflicts(newConflicts);
    setSilentUpdates(newSilent);
    setNewFromXero(newCandidates);
  } catch (err) {
    setReconcileError(err instanceof Error ? err.message : 'Reconciliation failed');
  } finally {
    setReconcileLoading(false);
  }
};

const applyReconciliation = () => {
  // 1. Apply silent updates (no operator interaction needed)
  for (const { memberId, update } of silentUpdates) {
    actions.updateTeamMember(memberId, update);
  }
  // 2. Apply per-field decisions for conflicts
  for (const memberDiff of conflicts) {
    const member = state.teamMembers.find(m => m.id === memberDiff.memberId);
    if (!member) continue;
    const decisions = pendingDecisions[memberDiff.memberId] ?? {};
    let memberUpdate: Partial<TeamMember> = { _xeroImportedAt: new Date().toISOString() };
    for (const fieldDiff of memberDiff.fields) {
      if (fieldDiff.verdict !== 'conflict') continue;
      const decision = decisions[fieldDiff.field] ?? 'keep-mine';  // default to keep-mine if undecided
      const partial = applyReconciliationDecision(member, fieldDiff.field, decision, fieldDiff.newXeroValue);
      memberUpdate = { ...memberUpdate, ...partial };
    }
    actions.updateTeamMember(memberDiff.memberId, memberUpdate);
  }
  // 3. Add selected new-from-Xero employees (same path as 52-01 import)
  const today = new Date();
  const cutoff = new Date(today.getTime() + 7 * 24 * 3600 * 1000);
  for (const emp of newFromXero) {
    if (!selectedNewIds.has(emp.employee_id)) continue;
    const enriched = enrichWizardMemberFromXeroEmployee(emp);
    const startDate = emp.start_date ? new Date(emp.start_date) : undefined;
    const isPlannedHire = startDate && startDate > cutoff;
    if (isPlannedHire) {
      actions.addNewHire({
        id: `hire-${Date.now()}-${emp.employee_id.slice(0, 6)}`,
        ...enriched,
        startMonth: `${startDate!.getFullYear()}-${String(startDate!.getMonth() + 1).padStart(2, '0')}`,
        salary: enriched.currentSalary ?? 0,
        superAmount: 0,
      } as any);
    } else {
      actions.addTeamMember({
        id: `team-${Date.now()}-${emp.employee_id.slice(0, 6)}`,
        ...enriched,
        increasePct: 0,
        newSalary: enriched.currentSalary ?? 0,
        superAmount: 0,
      } as any);
    }
  }
  // 4. Close modal + clear state
  setShowReconcile(false);
  setConflicts([]);
  setSilentUpdates([]);
  setNewFromXero([]);
  setSelectedNewIds(new Set());
  setPendingDecisions({});
};

// Bulk action handlers
const acceptAllXero = () => {
  const next: typeof pendingDecisions = {};
  for (const memberDiff of conflicts) {
    next[memberDiff.memberId] = {};
    for (const f of memberDiff.fields) {

};
const keepAllMine = () => {

    next[memberDiff.memberId] = {};

      if (f.verdict === 'conflict') next[memberDiff.memberId][f.field] = 'keep-mine';

  }

};

const inSync = !reconcileLoading && !reconcileError &&

              <h3 className="font-medium">Conflicts requiring your decision ({conflicts.length})</h3>
              <div className="flex gap-2">
                <button onClick={acceptAllXero} className="text-xs text-blue-600 underline">Accept all Xero changes</button>
                <button onClick={keepAllMine} className="text-xs text-gray-600 underline">Keep all my changes</button>
              </div>
            </div>
            {conflicts.map(memberDiff => {
              const member = state.teamMembers.find(m => m.id === memberDiff.memberId);
              if (!member) return null;
              return (
                <div key={memberDiff.memberId} className="border rounded p-3 mb-2">
                  <div className="font-medium mb-2">{member.name}</div>
                  {memberDiff.fields.filter(f => f.verdict === 'conflict').map(f => {
                    const decision = pendingDecisions[memberDiff.memberId]?.[f.field];
                    return (
                      <div key={f.field} className="flex items-center justify-between py-1 text-sm" data-conflict-field={f.field}>
                        <div>
                          <span className="font-mono text-xs text-gray-500">{f.field}</span>:
                          Xero now shows <strong>{String(f.newXeroValue)}</strong>;
                          you have <strong>{String(f.currentValue)}</strong>
                        </div>
                        <div className="flex gap-1">
                          <button
                            onClick={() => setPendingDecisions(p => ({ ...p, [memberDiff.memberId]: { ...(p[memberDiff.memberId] ?? {}), [f.field]: 'keep-mine' } }))}
                            aria-label={`Keep yours for ${f.field}`}
                            className={`px-2 py-1 text-xs rounded ${decision === 'keep-mine' ? 'bg-gray-200' : 'border'}`}
                          >Keep yours</button>
                          <button
                            onClick={() => setPendingDecisions(p => ({ ...p, [memberDiff.memberId]: { ...(p[memberDiff.memberId] ?? {}), [f.field]: 'accept-xero' } }))}
                            aria-label={`Accept Xero for ${f.field}`}
                            className={`px-2 py-1 text-xs rounded ${decision === 'accept-xero' ? 'bg-blue-200' : 'border'}`}
                          >Accept Xero</button>
                        </div>
                      </div>
                    );
                  })}
                </div>
              );
            })}
          </section>
        )}

        {newFromXero.length > 0 && (
          <section>
            <h3 className="font-medium mb-2">New from Xero ({newFromXero.length})</h3>
            <div className="text-xs text-gray-500 mb-2">Employees in Xero not yet in your forecast.</div>
            {newFromXero.map(emp => (
              <label key={emp.employee_id} className="flex items-center gap-2 py-1 text-sm">
                <input
                  type="checkbox"
                  aria-label={`Add ${emp.full_name}`}
                  checked={selectedNewIds.has(emp.employee_id)}
                  onChange={e => {
                    setSelectedNewIds(prev => {
                      const next = new Set(prev);
                      if (e.target.checked) next.add(emp.employee_id); else next.delete(emp.employee_id);
                      return next;
                    });
                  }}
                />
                {emp.full_name} — {emp.employment_type} — {emp.pay_frequency ?? '—'}
              </label>
            ))}
          </section>
        )}
      </div>
      <div className="flex justify-end gap-2 p-4 border-t">
        <button onClick={() => setShowReconcile(false)}>{inSync ? 'Close' : 'Cancel'}</button>
        {!inSync && (
          <button
            onClick={applyReconciliation}
            disabled={totalPendingChanges === 0}
            aria-label={`Apply ${totalPendingChanges} changes`}
          >
            Apply {totalPendingChanges} change{totalPendingChanges === 1 ? '' : 's'}
          </button>
        )}
      </div>
    </div>
  </div>
)}
```
