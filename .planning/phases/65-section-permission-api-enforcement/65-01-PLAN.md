    1. Accepts the caller's `supabase` client (a `SupabaseClient` from `@supabase/supabase-js`). **Does NOT call `createServiceRoleClient()`**. Queries flow through RLS.

    4. If all allow paths fall through, return `{ allow: false, reason: 'not_a_member', sectionKey }`.
